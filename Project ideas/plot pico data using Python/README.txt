This project gets sensor (ldr) data from Raspberry Pi Pico then plots it in realtime on a Python graph on a PC.

Code:
main.py - save on Pi Pico (code should run automatically when pluged to power source with USB cable)
python_pico_plot.py - save and run on PC


Requirements:

PC:
- Windows 10
- Python 3
- Numpy
- PySerial library
- Matplotlib library

Raspberry Pi Pico:
- MicroPython
- Micro USB connector
- Photosensor
- 10K resistor
- Jumper wires
